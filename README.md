# COSC360-Group-6

**Steps to run through docker:**<br>
Create a .env file in backend/ and paste the variables

Open a new terminal
In the root folder
- docker-compose up --build

<br>

**Steps to run locally:**<br>
Create a .env file in backend/ and paste the variables

Open a new terminal
Enter backend directory - cd backend
 - npm install
 - npm run dev

Open a new terminal
Enter frontend directory - cd frontend
 - npm install
 - npm run dev
